Android 点击头像拍照或者从图库中选择

发表于 2016-04-30   |     |   阅读次数

现在基本上每个app都会有用户设置界面,都会有头像的设置:
其实这就是涉及Android的拍照功能,Android拍照分为调用系统拍照和自定义拍照,今天我们来一起学习下使用系统拍照实现设置头像的功能.

如图:

基础讲解

在使用系统相机的时候,就是要去调用系统应用,这样我们就需要使用隐式Intent来调用系统相机:

  1. 拍照

    android.media.action.IMAGE_CAPTURE

  2. 相册

    android.intent.action.GET_CONTENT

  3. 裁剪

    com.android.camera.action.CROP

  4. 调用相机的功能需要调用intent put参数的形式来客制化裁剪

    intent.putExtra("crop", "true");// 发送裁剪信号
intent.putExtra("aspectX", 1);// 裁剪比例
intent.putExtra("aspectY", 1);
intent.putExtra("outputX", 600);// 输出大小
intent.putExtra("outputY", 600);
intent.putExtra("noFaceDetection", true); // 关闭人脸识别
intent.putExtra("scale", true);// 缩放
intent.putExtra("scaleUpIfNeeded", true);// 去黑边
intent.putExtra("return-data", false);// 不返回缩略图
intent.putExtra(MediaStore.EXTRA_OUTPUT, url地址);//输出文件
intent.putExtra("outputFormat", Bitmap.CompressFormat.PNG.toString());//输出格式

    这些参数不是必须使用的,需要什么属性就使用什么属性.

代码实现

  1. 首先,我们需要定义2个uri,用来存储拍照后的照片和裁剪的照片.创建图片的File

    private Uri photoUri;
private File picFile;

private Uri tempUri;
private File tempPicFile;

  2. 添加点击事件

    @Override
public void onClick(View view) {
    switch (view.getId()) {
        case R.id.btn_photograph:
            doHandlerPhoto(PIC_FROM_CAMERA);// 从照相机获取
            break;

        case R.id.btn_gallery:
            doHandlerPhoto(PIC_FROM＿LOCALPHOTO);// 从相册中去获取
            break;

        case R.id.btn_cancle:
            dismiss();
            break;
    }
}

  3. 设置裁剪的公共参数,裁剪的输出为tempUri

    /**
 * 设置公用参数
 */
private void setIntentParams(Intent intent) {
    intent.putExtra("crop", "true");
    intent.putExtra("aspectX", 1);// 裁剪比例
    intent.putExtra("aspectY", 1);
    intent.putExtra("outputX", 600);// 输出大小
    intent.putExtra("outputY", 600);
    intent.putExtra("noFaceDetection", true); // 关闭人脸识别
    intent.putExtra("scale", true);// 缩放
    intent.putExtra("scaleUpIfNeeded", true);// 去黑边
    intent.putExtra("return-data", false);// 不返回缩略图
    intent.putExtra(MediaStore.EXTRA_OUTPUT, tempUri);//输出文件
    intent.putExtra("outputFormat", Bitmap.CompressFormat.PNG.toString());//输出格式
}

  4. 拍照并保存到FileUri

    Intent cameraIntent = new Intent(MediaStore.ACTION_IMAGE_CAPTURE);
cameraIntent.putExtra(MediaStore.EXTRA_OUTPUT, photoUri);
mContext.startActivityForResult(cameraIntent, PIC_FROM_CAMERA);

  5. 在onActivityResult中调用裁剪

    /**
 * 启动裁剪
 */
private void cropImageUriByTakePhoto() {
    Intent intent = new Intent("com.android.camera.action.CROP");
    intent.setDataAndType(photoUri, "image/*");
    setIntentParams(intent);
    mContext.startActivityForResult(intent, PIC_FROM＿LOCALPHOTO);
}

  6. 当点击相册时调用图片剪辑程序

    Intent intent = getCropImageIntent();
mContext.startActivityForResult(intent, PIC_FROM＿LOCALPHOTO);

/**
 * 调用图片剪辑程序
 */
public Intent getCropImageIntent() {
    Intent intent = new Intent(Intent.ACTION_GET_CONTENT, null);
    intent.setType("image/*");
    setIntentParams(intent);
    return intent;
}

  7. 最终不管拍照还是相册,裁剪之后都会调回到onActivityResult的PIC_FROM＿LOCALPHOTO,在这里面我们将图片设置到ImageView上,还可以做一些其他的操作,比如说上传图片之类的.

  8. 在实际项目中出现的问题: 在一些调用系统相机,相机横屏的时候,会出现拍照后不能调用裁剪的情况,这是由于系统转屏销毁的原因,我们需要在AndroidManifest.xml中添加:

    <activity
    android:name=".MainActivity"
    android:configChanges="orientation|keyboardHidden|screenSize"/>
